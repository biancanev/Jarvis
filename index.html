What is Jarvis?

Jarvis is a Windows based program that acts as an A.I. for a windows computer. Some scripts include VBScript and Batch File, which can be dangerous for a computer, as they can change files on a computer. Be careful of malware that can pose as Jarvis. Jarvis is currently at v.1.0.2, and is currently stable. You cannot voice activate Jarvis(with a term like "Hey Jarvis"). Jarvis require antivirus filters to be turned off to install, due to VBScripting.
