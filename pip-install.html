Install Jarvis

Jarvis v.1.0.2
Jarvis v.1.0.1 (LTE)

Jarvis is currently only supported on Windows OS. In order to run this on Linux OS, please install Wine

How to install

Step 1: Download

Click one of the links above

Make sure to turn your anti-virus sofware off.

Step 2: File Directory

Once you have clicked a link, a popup box should pop up asking what you would like to do with Jarvis_Install.zip click Save As. Then save to C:\Users\%USERNAME%\Desktop

Step 3: Setup for Usage

Once you have saved Jarvis to your desktop, it's time for the computer to get used to the sound of your voice. The first thing you have to do is unzip the files you have just downloaded. Right click on the .zip file and click Extract All. A popup message should appear. Click the browse button and select Desktop. Click Extract, then wait for the computer to finish unzipping the file.

Step 4: Voice Callibration

Double click the file called Jarvis or Jarvis_v_1_0_2.vbs. Follow the instructions on the screen.

Warning: Memory(RAM) utilization might go up to 100% in legnthy commands. If your computer seems to freeze up, check the task manager.
